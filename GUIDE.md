# CRUD CON EXPRESS - TYPEORM - TYPESCRIPT

## Pasos

### Instalar como dependencia de desarrollo typescript

    npm install typescript -D

### Crear el compilador

    npx tsc --init

    En el archivo tsconfig.json indicar el punto de entrada y de salida del compilador

### Para facilitar el desarrollo

    npm install ts-node-dev -D

    Esto es como nodemon para typescript

### Instalar los tipados

    npm install -D @types/express

### Instalar TypeORM

    npm install --save typeorm 
    o
    npm install typeorm --legacy-peer-deps
